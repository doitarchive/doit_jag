2.2 Memory Track


Das Jaguar Memory Pack



Das Memory Modul sollte man sich direkt zulegen wenn man das erste Jaguar Spiel auf CD-ROM kauft, bekannterweise lassen sich ja auf einer CD keine Spielstände speichern.

Der sog. "Memory Track" von Atari ist in der Lage per internem EEprom 125.000 Bytes zu speichern. In der Praxis bedeutet das da Sie damit in der Lage sind von jedem CD-ROM Spiel das Sie gekauft haben mehrere verschiedene Spielstände zu sichern. Die Informationen gehen nicht verloren auch wenn das Modul aus dem CD-ROM Laufwerk herrausgezogen ist und auch länger nicht benutzt wird.

Die Speicherkapazität sollte laut Hersteller für weit über 200 verschiede Spiele aussreichen!

Es werden in diesem Modul neben den eigentlichen Highscores auch komplette Scenarien wie z.B bei MYST hinterlegt. Einzelne Spiellevels können ebenso gesichert werden so daß Sie nicht wieder bei 0 beginnen müssen.

Wenn Sie das Memory Modul in der Konsole bzw. im CD-ROM stecken haben können Sie immer beim Einschalten der Jagaur-Konsole mit gedrückter "Option"-Taste den Memory Manager aktivieren.

Vorsicht: Hier können Sie auch Ihre Spielstände löschen!!
