Jaguar - Schnittstellen


Die Steckbuchsen für die Joypads, die Steuersignale der Joypads werden im JAGUAR über Treiberbausteine geführt. I.d.R 74HC244 und 47HC357 in SMD-Technik. Bei einem defekt bzw. dann wenn keine Steuerbefehle von JAGUAR angenommen werden sollte man sich mit diesen Bausteinen näher beschäftigen.

Gleiches trifft für den Modulslot zu. Wobei hier jedoch nicht alle Signale über Treiberbausteine geführt wurden.

Man sollte so gut es geht in jedem Fall vermeiden irgendwas an den JAGUAR An- oder Abzustecken solange das Gerät eingeschaltet ist. Eine Reparatur ist hier möglich, wird aber durch die SMD-Technik erschwert.

Die meisten Signalleitungen sind über Entstörspulen oder kleinen Schutzwiderständen mit den von aussen zugänglichen Kontakten verbunden. Eindringlich können die Leute gewarnt werden die sich selber einen Joystick oder sonstiges Zubehör für diese Maschine selber bauen wollen.
In der englischen JAGUAR.FAQ finden Sie einen Schaltplan in ASCII Darstellung der das Funktionsprinzip des JAGUAR-Controllers zeigt.

Bis heute konnt ich nicht feststellen das diese Schaltung falsch ist.
Man könnte sich also daran Orientieren.

The most wanted complete VideoPort description 


Component Side A               Solderside B

Pin  1A = Audio Left           Pin  1B = Audio Right
Pin  2A = Audio GND            Pin  2B = Audio GND
Pin  3A = NC                   Pin  3B = Video GND
Pin  4A = Video GNG            Pin  4B = RED (RGB)
Pin  5A = BLUE (RGB)           Pin  5B = VSL (V-SYNC)
Pin  6A = HSL1 (H-SYNC)        Pin  6B = NC
Pin  7A = GREEN (RGB)          Pin  7B = VIDEO GND
Pin  8A = S_COLOR              Pin  8B = S_LUMA
Pin  9A = NC                   Pin  9B = GND via C48 (CS)
Pin 10A = INC                  Pin 10B = Video GND
Pin 11A = +9V via R83 (100 R)  Pin 11B = COMPOSITE VIDEO
Pin 12A = NC                   PIN 12B = NC

The most importend thing:
The complete Pinout of the incredible DSP-Port 

Component Side A               Solder Side B

Pin 1A = U15 GND               Pin 1B = Vcc +5Volt
Pin 2A = SCK                   Pin 2B = UART0
Pin 3A = UW                    Pin 3B = UART1
Pin 4A = TxD                   Pin 4B = NC
Pin 5A = RxD                   Pin 5B = NC
Pin 6A = U15 GND               Pin 6B = GND (Shild)

All information collected from schematic diagram of 23 September 1993
Watch out while connecting somthing!
No fuses or some other safty equipment on ANY Pin avaiable!!
